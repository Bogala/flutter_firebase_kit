# Product Roadmap

## Phase 1: MVP

- Clean Architecture skeleton — `core/`, `data/`, `domain/`, `ui/` layers with `*_module.dart` exports
- Dependency injection — `get_it` + `injectable` with flavor-scoped singletons
- BLoC feature template — example feature showing page/view/bloc/interactor/module structure
- BDD/Gherkin test infrastructure — pre-installed step definitions, test scaffold, mock interceptor
- 6-flavor setup — `prod`, `preprod`, `recette`, `integration`, `dev`, `test` with dedicated entry points
- Network layer — Dio + Retrofit with caching interceptor and mock interceptor for tests
