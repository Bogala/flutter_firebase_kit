# Commit Format

```
<JIRA-ticket> - <type>(<scope>) : <short summary in French>
```

**Types:** `build` `ci` `docs` `feat` `fix` `perf` `refactor` `test`

**Scopes:** `core` `data` `domain` `ui` `design`

**Example:**
```
LIS-123 - feat(ui) : ajouter écran de connexion
```

- JIRA ticket prefix is always required — no exceptions
- Summary is always in French
- One space before and after ` : `
